# 嘸蝦米輸入法 OSX

## 情境
1. 想超越原本嘸蝦米，享受自由度更高的自定義`字根檔`

2. 有時只是暫時使用別人的電腦，但上面沒有安裝嘸蝦米，要到官網下載安裝都覺得麻煩

> 無腦一條指令安裝，給那些跟我一樣沒有嘸蝦米就活不下去的蝦米族 :)

## 安裝
    curl https://raw.githubusercontent.com/rwu823/liu/master/install.sh | sh
